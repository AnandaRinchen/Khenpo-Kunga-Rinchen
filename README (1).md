# Khenpo Kunga Rinchen — Personal Profile

A beautifully designed biographical page for Khenpo Kunga Rinchen, distinguished Khenpo of the Sakya tradition and compiler of the 116-volume Gyud-de Kuntu.

## 🚀 How to Deploy on GitHub Pages

### Option A: As Your Main Site (`yourusername.github.io`)

1. Create a new repository named **exactly** `yourusername.github.io`
   (replace `yourusername` with your actual GitHub username)
2. Upload `index.html` to the repository root
3. Go to **Settings → Pages → Source** → select `main` branch
4. Your site will be live at `https://yourusername.github.io`

### Option B: As a Project Site (`yourusername.github.io/profile`)

1. Create a new repository with any name (e.g., `profile`)
2. Upload `index.html` to the repository root
3. Go to **Settings → Pages → Source** → select `main` branch
4. Your site will be live at `https://yourusername.github.io/profile`

## 📁 Files

## 🔗 Custom Domain (Optional)

To use a custom domain like `www.kungarinchen.com`:

1. Go to **Settings → Pages → Custom domain**
2. Enter your domain name
3. Add a `CNAME` record with your DNS provider pointing to `yourusername.github.io`
4. Check "Enforce HTTPS"
